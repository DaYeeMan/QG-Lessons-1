Introduction

Return probabilities in Markov chains measure the likelihood that the chain will return to a given state after a certain number of steps. These probabilities are fundamental for understanding the long-term behavior and classification of states.

Key Concepts & Formulas

  • n-step Return Probability: The probability of returning to state i in exactly n steps:
    \[ f_{ii}^{(n)} = P(X_n = i, X_k \neq i \text{ for } 1 \leq k < n | X_0 = i) \]
  • Return Probability: The probability of ever returning to state i:
    \[ f_{ii} = \sum_{n=1}^{\infty} f_{ii}^{(n)} \]
  • Expected Return Time: The expected number of steps to return to state i:
    \[ \mu_{ii} = \sum_{n=1}^{\infty} n f_{ii}^{(n)} \]
  • Recurrence: A state is recurrent if f_{ii} = 1, transient if f_{ii} < 1.

Worked Example

Consider a simple random walk on the integers with p = 0.5. For state 0:

\[ f_{00}^{(2n)} = \binom{2n}{n} \left(\frac{1}{2}\right)^{2n} \]

The return probability is:

\[ f_{00} = \sum_{n=1}^{\infty} f_{00}^{(2n)} = 1 \]

This shows that state 0 is recurrent (the walk will return to 0 with probability 1).

Practice Problems

Problem 1

For a biased random walk with p > 0.5, show that the return probability to state 0 is less than 1.

For p > 0.5, the walk has a drift to the right. The return probability is f_{00} = 2(1-p) < 1, making state 0 transient.



Problem 2

What is the relationship between return probabilities and the stationary distribution?

For a positive recurrent state i, the stationary probability π_i = 1/μ_{ii}, where μ_{ii} is the expected return time.



Problem 3

How do you compute return probabilities for a finite irreducible Markov chain?

For finite irreducible chains, all states are recurrent. Return probabilities can be computed using the fundamental matrix or by solving the system of equations for hitting probabilities.
