Introduction

Reliability theory deals with the probability that a system or component will perform its intended function for a specified period under given conditions. It is essential in engineering, quality control, and risk assessment.

Key Concepts & Formulas

  • Reliability Function: R(t) = P(T > t), where T is the time to failure. For exponential distribution:
    \[ R(t) = e^{-\lambda t} \]
  • Hazard Rate: h(t) = f(t)/R(t), the instantaneous failure rate at time t.
  • Mean Time to Failure (MTTF): E[T] = ∫₀^∞ R(t) dt.
  • Series System: For n components in series, R_system(t) = ∏ᵢ Rᵢ(t).
  • Parallel System: For n components in parallel, R_system(t) = 1 - ∏ᵢ (1 - Rᵢ(t)).
  • Weibull Distribution: A flexible distribution for modeling failure times with shape parameter β and scale parameter η.

Worked Example

Consider a system with two components in series, each with exponential failure times with rates λ₁ = 0.01 and λ₂ = 0.02 per hour. The system reliability at t = 50 hours is:

\[ R_{system}(50) = e^{-0.01 \cdot 50} \cdot e^{-0.02 \cdot 50} = e^{-0.5} \cdot e^{-1} = e^{-1.5} \approx 0.223 \]

This shows how component failures multiply in series systems.

Practice Problems

Problem 1

For a parallel system with two identical components each having reliability 0.8, find the system reliability.

R_system = 1 - (1-0.8)² = 1 - 0.04 = 0.96.



Problem 2

Explain the relationship between hazard rate and reliability function.

The hazard rate h(t) = -d/dt[ln(R(t))], and R(t) = exp(-∫₀^t h(s) ds). The hazard rate describes the instantaneous failure intensity.



Problem 3

How do you model systems with both series and parallel components?

Use reliability block diagrams and decompose the system into series and parallel subsystems, then combine using the appropriate formulas for each configuration.
