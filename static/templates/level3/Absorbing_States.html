Introduction

Absorbing states are states that, once entered, cannot be left. Markov chains with absorbing states are used to model processes that eventually terminate or are "trapped" in certain states.

Key Concepts & Formulas

  • Absorbing State: State i is absorbing if P_{ii} = 1 and P_{ij} = 0 for j â‰  i.
  • Absorbing Markov Chain: A chain with at least one absorbing state and from every state there is a nonzero probability of reaching an absorbing state.
  • Fundamental Matrix: For the submatrix Q of transient states:
    \[ N = (I - Q)^{-1} \]
    N_{ij} gives the expected number of visits to state j starting from i before absorption.
  • Absorption Probabilities: The probability of being absorbed in a particular absorbing state can be computed using the fundamental matrix and the R matrix of transitions from transient to absorbing states.
  • Expected Time to Absorption: The expected number of steps before absorption starting from state i is the sum of the ith row of N.

Worked Example

Consider a Markov chain with transition matrix:

\[ P = \begin{pmatrix} 1 & 0 & 0 \\ 0.5 & 0.5 & 0 \\ 0.2 & 0.3 & 0.5 \end{pmatrix} \]

State 1 is absorbing. The submatrix Q for transient states is:

\[ Q = \begin{pmatrix} 0.5 & 0 \\ 0.3 & 0.5 \end{pmatrix} \]

The fundamental matrix is N = (I-Q)^{-1}.

Practice Problems

Problem 1

For the above chain, compute the expected number of steps to absorption starting from state 2.

The expected number of steps is the sum of the second row of N. Compute N = (I-Q)^{-1} and sum the entries in the second row.



Problem 2

How do you compute the probability of being absorbed in a particular absorbing state?

The absorption probabilities are given by the matrix B = NR, where R is the matrix of transition probabilities from transient to absorbing states.



Problem 3

Give an example of a real-world process modeled by an absorbing Markov chain.

Board games like Monopoly, where the game ends when a player goes bankrupt, can be modeled as absorbing Markov chains. The bankrupt state is absorbing.
