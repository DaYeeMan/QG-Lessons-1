Introduction

First passage times in Markov chains refer to the time it takes for the chain to reach a particular state for the first time, starting from a given initial state. This concept is crucial for understanding hitting probabilities and absorption times.

Key Concepts & Formulas

  • First Passage Time: Let T_{ij} be the first time the chain reaches state j starting from state i:
    \[ T_{ij} = \min\{n \geq 1 : X_n = j \text{ given } X_0 = i\} \]
  • Expected First Passage Time: The expected time to reach state j from state i:
    \[ \mu_{ij} = E[T_{ij}] \]
  • Hitting Probability: The probability of ever reaching state j from state i:
    \[ f_{ij} = P(T_{ij} < \infty) \]
  • Recurrence Time: When i = j, T_{ii} is the recurrence time (time to return to state i).

Worked Example

Consider a 3-state Markov chain with transition matrix:

\[ P = \begin{pmatrix} 0.5 & 0.3 & 0.2 \\ 0.2 & 0.6 & 0.2 \\ 0.1 & 0.3 & 0.6 \end{pmatrix} \]

To find the expected first passage time from state 1 to state 3:

\[ \mu_{13} = 1 + 0.5\mu_{13} + 0.3\mu_{23} \]

This gives a system of equations that can be solved for the expected passage times.

Practice Problems

Problem 1

For a simple random walk on {0,1,2} with absorbing boundaries, find the expected time to absorption starting from state 1.

Let μ₁ be the expected time from state 1. Then: μ₁ = 1 + 0.5μ₀ + 0.5μ₂ = 1 + 0.5(0) + 0.5(0) = 1, since μ₀ = μ₂ = 0 (already absorbed).



Problem 2

What is the relationship between first passage times and the fundamental matrix of a Markov chain?

The fundamental matrix N = (I - Q)⁻¹ gives the expected number of visits to transient states before absorption. The expected first passage times can be computed using N.



Problem 3

How do you compute hitting probabilities for a finite Markov chain?

Hitting probabilities satisfy the system: f_{ij} = P_{ij} + Σ_{k≠j} P_{ik} f_{kj}, with boundary conditions f_{jj} = 1 and f_{ij} = 0 if j is unreachable from i.
