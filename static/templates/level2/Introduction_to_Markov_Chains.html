Introduction

A Markov chain is a stochastic process that moves between states with probabilities that depend only on the current state (not the past). Markov chains are widely used in finance, physics, and computer science.

Key Concepts & Formulas

  • Markov Property: The future is independent of the past given the present:
    \[ P(X_{n+1} = j | X_n = i, X_{n-1}, ..., X_0) = P(X_{n+1} = j | X_n = i) \]
  • Transition Probability: The probability of moving from state i to state j in one step is \( P_{ij} \).
  • State Space: The set of all possible states.

Worked Example

Consider a simple weather model: if today is sunny, tomorrow is sunny with probability 0.8 and rainy with 0.2. If today is rainy, tomorrow is sunny with 0.4 and rainy with 0.6.

\[ P = \begin{pmatrix} 0.8 & 0.2 \\ 0.4 & 0.6 \end{pmatrix} \]

This is the transition matrix for the Markov chain.
