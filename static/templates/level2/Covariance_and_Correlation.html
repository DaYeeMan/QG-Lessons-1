Introduction

Covariance and correlation measure the relationship between two random variables. Covariance indicates the direction of the linear relationship, while correlation standardizes this measure, making it easier to interpret.

Key Concepts & Formulas

  • Covariance: \( \operatorname{Cov}(X, Y) = E[(X - E[X])(Y - E[Y])] \)
  • Correlation: \( \rho_{X,Y} = \frac{\operatorname{Cov}(X, Y)}{\sigma_X \sigma_Y} \)
  • \( -1 \leq \rho_{X,Y} \leq 1 \)

Worked Example

Suppose we have two random variables X and Y with the following paired values:

X: [1, 2, 3, 4, 5]
Y: [2, 4, 6, 8, 10]

Covariance: The covariance is positive, indicating X and Y increase together.
Correlation: Since Y is a perfect linear function of X, the correlation is 1.

Practice Problems

Problem 1

Given X = [1, 2, 3], Y = [4, 5, 6], calculate the covariance and correlation.

Cov(X, Y) = E[XY] - E[X]E[Y] = 1, Correlation = 1 (since Y is a linear function of X).



Problem 2

Explain why correlation is a better measure than covariance for comparing relationships between different pairs of variables.

Correlation is dimensionless and always between -1 and 1, making it easier to compare across different variable pairs, while covariance depends on the units of X and Y.



Problem 3

Can two variables have zero covariance but not be independent? Give an example.

Yes. For example, if X is uniformly distributed on [-1, 1] and Y = X^2, then Cov(X, Y) = 0 but X and Y are not independent.
