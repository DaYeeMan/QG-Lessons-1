Introduction

The Transition Rate Matrix (also called the infinitesimal generator matrix) is the fundamental mathematical object that characterizes a continuous-time Markov chain. It contains the rates at which the process transitions between states and determines all the probabilistic behavior of the CTMC.

Key Concepts & Formulas

  • Definition: For a CTMC with state space S, the transition rate matrix Q is defined by:
    \[ q_{ij} = \lim_{h \to 0} \frac{P(X(h) = j | X(0) = i)}{h}, \quad i \neq j \]
  • Diagonal Elements: The diagonal elements satisfy:
    \[ q_{ii} = -\sum_{j \neq i} q_{ij} \]
  • Properties:
    • \(q_{ij} \geq 0\) for \(i \neq j\)
    • \(q_{ii} \leq 0\)
    • \(\sum_{j} q_{ij} = 0\) for all i
  • Sojourn Time: Time spent in state i is exponential with rate \(-q_{ii}\).
  • Transition Probabilities: The probability of transitioning from i to j (given a transition occurs) is:
    \[ \frac{q_{ij}}{-q_{ii}} \]

Worked Example

Consider a CTMC with three states representing a machine that can be working (0), under repair (1), or failed (2). The transition rate matrix is:

\[ Q = \begin{pmatrix} -0.1 & 0.08 & 0.02 \\ 0.5 & -0.5 & 0 \\ 0.2 & 0 & -0.2 \end{pmatrix} \]

Interpretation:

  • From working state: breakdown at rate 0.1 (0.08 to repair, 0.02 to failed)
  • From repair state: fixed at rate 0.5
  • From failed state: recovery at rate 0.2

Practice Problems

Problem 1

For the matrix Q above, what is the expected time the machine spends in the working state before transitioning?

The sojourn time in state 0 is exponential with rate \(-q_{00} = 0.1\), so the expected time is \(1/0.1 = 10\) time units.



Problem 2

Given a transition occurs from state 0, what is the probability it goes to state 1?

\( \frac{q_{01}}{-q_{00}} = \frac{0.08}{0.1} = 0.8 \)



Problem 3

Verify that the sum of each row in Q equals zero.

Row 0: \(-0.1 + 0.08 + 0.02 = 0\)
Row 1: \(0.5 - 0.5 + 0 = 0\)
Row 2: \(0.2 + 0 - 0.2 = 0\)
